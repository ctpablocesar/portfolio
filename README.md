## 🚀 Instalación y Uso

1. **Clona el repositorio**

   ```bash
   git clone https://github.com/ctpablocesar/portfolio.git
   cd portfolio
   ```

2. **Instala las dependencias**

   ```bash
   pnpm install
   ```

3. **Inicia el servidor de desarrollo**

   ```bash
   pnpm dev
   ```

4. **Abre en el navegador**
   ```
   http://localhost:3000
   ```
